# Healthcare Patient Management App

## 🚀 Project Purpose

The goal of this project is to simplify and automate the healthcare appointment booking process, making it more efficient for both patients and healthcare administrators. By reducing manual tasks, minimizing errors, and providing real-time updates, the app enhances the patient experience while improving operational efficiency for administrators.

## 🔋 Key Features

- *Patient Registration*  
   Patients can quickly sign up, create profiles, and manage their information securely within the app.
   
- *Appointment Scheduling*  
   Patients can easily book new appointments with doctors at their convenience, choosing preferred times and doctors. Multiple appointments can be scheduled, providing a hassle-free experience for patients needing ongoing care.

- *Admin Appointment Management*  
   Administrators can view, confirm, or reschedule appointments efficiently using a simple and intuitive dashboard. Admins can also cancel appointments when needed, ensuring real-time control over the scheduling process.

- *SMS Notifications*  
   Automated SMS notifications are sent to patients upon appointment confirmation, providing timely reminders and reducing no-show rates.

- *File Uploads*  
   Patients can securely upload and store important documents (e.g., medical records) using *Appwrite* storage, providing a seamless way to manage patient data.

- *Real-Time Performance Monitoring*  
   *Sentry* is used to track app performance, detect errors, and ensure the app remains reliable and bug-free.

- *Complete Responsiveness*  
   The application is fully responsive, providing a smooth user experience across all devices (mobile, tablet, desktop).

## ⚙️ Tech Stack

- *Next.js*: Used for fast, scalable, and SEO-friendly server-side rendering.
- *Appwrite*: Backend services and file storage, simplifying authentication and database management.
- *Typescript*: For static typing, improving code quality and maintainability.
- *TailwindCSS*: Utility-first CSS framework for rapid and responsive UI design.
- *Twilio*: Integrated for SMS notifications to patients.
- *Sentry*: Performance monitoring and real-time error tracking.
